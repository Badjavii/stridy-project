# Stridy

## Acerca de este proyecto

Stridy es un espacio de trabajo integral para la productividad académica, diseñado para estudiantes universitarios. Centraliza instituciones, asignaturas, calificaciones, calendario y horario en una sola plataforma, con un asistente de IA que comprende tu contexto académico y te ayuda a estudiar de forma más eficaz.

## Características

- **Calculadora de calificaciones:** Calcula automáticamente la puntuación necesaria para aprobar cada asignatura en las evaluaciones pendientes.

- **Asistente de IA:** Asistente de chat académico con tecnología OpenRouter que conoce tus asignaturas, calificaciones y apuntes.

- **Generador de exámenes con IA:** Pega cualquier contenido y obtén preguntas de práctica de opción múltiple al instante.

- **Horario semanal:** Horario visual con detección automática de conflictos.

- **Calendario mensual:** Gestiona exámenes, fechas límite y clases en un solo lugar.

- **Sistema social:** Agrega amigos y comparte calendarios y horarios.

- **Privacidad ante todo:** Sin correo electrónico ni contraseña: acceso con una clave secreta de 16 caracteres.

## Tecnologías

- **Frontend:** React + Vite + Módulos CSS
- **Backend:** Bun + Hono
- **Base de datos:** PostgreSQL
- **IA:** API de OpenRouter (modelos gratuitos)
- **Despliegue:** DokPloy en VPS
- **Autenticación:** JWT + Claves de acceso estilo Mullvad
